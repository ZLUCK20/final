# 大学生期末刷题小工具 — 设计规格

## 概述

单页 HTML 刷题工具，支持单选/填空/判断题的随机抽题练习。深色护眼主题，纯前端，零依赖。

## 技术方案

- **形式**：单 HTML 文件（CSS + JS 全部内嵌）
- **数据**：题目以 JSON 格式嵌入页面底部 `<script>` 块，用户提供文档后手动转换嵌入
- **存储**：`localStorage` 保存错题记录
- **风格**：深色主题，简洁卡片式 UI

## 数据格式

```json
{
  "subject": "大学物理",
  "questions": [
    {
      "id": 1,
      "type": "单选",
      "question": "光的折射定律是指？",
      "options": ["反射角等于入射角", "折射光线与入射光线分居法线两侧", "光速不变原理", "以上都不是"],
      "answer": "B",
      "explain": "根据斯涅耳定律，折射光线与入射光线分居法线两侧..."
    },
    {
      "id": 2,
      "type": "判断",
      "question": "光在真空中的速度是 3×10⁸ m/s",
      "answer": "对",
      "explain": "光速在真空中恒为 c ≈ 3×10⁸ m/s"
    },
    {
      "id": 3,
      "type": "填空",
      "question": "牛顿第___定律阐述了力与加速度的关系",
      "answer": ["二", "2", "第二"],
      "explain": "牛顿第二定律 F=ma..."
    }
  ]
}
```

- **单选**：`options` 为 4 个选项，`answer` 为 A/B/C/D
- **判断**：无 `options`，`answer` 为 "对" 或 "错"
- **填空**：无 `options`，`answer` 为可接受的答案数组（模糊匹配，任意一个匹配即正确）

## 界面结构（共 4 个视图）

### 1. 设置页（首页）
- 科目标题 + 题目总数
- 抽题数量输入框（默认 10 题，可修改）
- 「开始答题」按钮
- 「错题本」按钮（有错题时显示角标数字）
- 「清空错题」按钮（二次确认）

### 2. 答题页
布局：
```
顶部：进度条 + 当前题号/总题数 + 实时正确率
中间：题目卡片（题型标签 + 题干 + 选项/输入框/判断按钮）
底部：导航按钮（上一题/下一题） + 提交/交卷按钮
反馈：提交后答案正确（绿色高亮）/错误（红色高亮 + 显示正确答案）
```

各题型交互：
- **单选**：点击选项卡片选中，再次点击可取消
- **判断**：两个大按钮「对」「错」，点击切换选中态
- **填空**：文本输入框 + 提交按钮

### 3. 结果页
- 大号分数：`答对 12/15 题 · 正确率 80%`
- 每题结果列表：题号 + 对错图标 + 正确答案 + 解析
- 「再来一轮」→ 回到设置页
- 「错题本」→ 错题页面

### 4. 错题本
- 列出所有错题（从 localStorage 读取）
- 每道题可展开查看解析
- 「重做错题」按钮 → 对错题随机排序开始练习
- 「清空错题本」按钮

## 功能逻辑

### 随机抽题
- 从题库中随机抽取用户指定数量，不重复
- 如果抽题数 > 题库总数，则全部抽取

### 提交答案
- 每次切换题目时自动保存当前题目的作答状态到内存
- 每题提交后立即显示对错反馈
- 填空题模糊匹配：去掉首尾空格后比较（用户输入 === answer 数组中任意一项）

### 错题记录
- 提交答案后若答错，自动将该题加入错题本（localStorage 持久化）
- 错题记录格式：`{ questionId: number, userAnswer: string, timestamp: number }`
- 答题界面中已答错的题目会显示错题标记（❗）
- 下次打开页面时错题本数据保留

## 样式约束

- 深色主题：背景 #1a1a2e 或类似深蓝黑，卡片 #16213e，强调色 #0f3460 或 #e94560
- 正确反馈：绿色 #2ecc71
- 错误反馈：红色 #e74c3c
- 字体：系统无衬线字体
- 移动端自适应
