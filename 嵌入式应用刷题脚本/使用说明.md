# 📚 期末刷题小工具 — 使用说明

## 文件结构

```
刷题脚本/
├── index.html              ← 主程序（双击打开，内建嵌入式题库）
├── 使用说明.md              ← 本文件
│
├── 嵌入式题库.json          ← 嵌入式题库文件（已直接内建在网页中）
├── 物联网工程.json          ← 物联网课后选择题（可选导入）
├── DSP_F28335.json         ← DSP F28335 填空题（可选导入）
└── docs/
    ├── specs/              ← 设计文档
    └── plans/              ← 实现计划
```

---

## 🚀 快速开始

**1. 打开网页**

双击 `index.html`，用浏览器打开即可使用，无需安装任何东西。

**2. 开始刷题（无需导入）**

打开后默认已加载 **嵌入式基础复习题库（76道单选+填空）**，可直接使用。

如需刷其他科目，点击左侧的 **「📤 导入题库」** → 选择对应的 `.json` 文件（如 `物联网工程.json`、`DSP_F28335.json`）。导入后页面会自动切换到新题库。

**3. 开始刷题**

- 设置抽题数量（默认 10 题）
- 点击 **「🚀 开始答题」**
- 作答后点 **「✅ 提交答案」** 看对错
- 每道题都有 **解析**
- 全部答完自动出 **成绩单**

**4. 错题本**

- 做错的题 **自动记录**
- 可单独 **重做错题**
- 支持 **逐题移除** 或 **一键清空**

## ✏️ 如何自己添加题目

### 方法一：直接编辑 JSON

用记事本打开 `.json` 文件，按以下格式添加：

```json
{
  "subject": "科目名",
  "questions": [
    {
      "id": 1,
      "type": "单选",
      "question": "题目内容？",
      "options": ["A选项", "B选项", "C选项", "D选项"],
      "answer": "A",
      "explain": "解析内容"
    },
    {
      "id": 2,
      "type": "判断",
      "question": "判断对错？",
      "answer": "对",
      "explain": "解析内容"
    },
    {
      "id": 3,
      "type": "填空",
      "question": "答案填在____里",
      "answer": ["正确答案1", "答案2"],
      "explain": "解析内容"
    }
  ]
}
```

支持三种题型：

| type | 说明 | answer 格式 |
|------|------|------------|
| `"单选"` | 单选题 | `"A"` / `"B"` / `"C"` / `"D"` |
| `"判断"` | 判断题 | `"对"` 或 `"错"` |
| `"填空"` | 填空题 | 可接受答案数组，任意匹配即算对（如 `["二","2","第二"]`） |

### 方法二：提供文档给我

把 Word 或 PDF 文档发给我，我帮你转换成 JSON。

## 📝 小技巧

- **填空题** 支持模糊匹配（忽略大小写和首尾空格）
- **回车键** 可以直接提交答案
- 打开网页即内置 **嵌入式基础复习题库**，无需手动导入
- 左侧可切换/删除题库，或导入其他 `.json` 题库文件
- 深色护眼主题，手机也能用

## ⚙️ 技术说明

- 数据存在浏览器本地（localStorage），关闭页面不丢失错题记录
- 所有代码内嵌在 `index.html` 中
