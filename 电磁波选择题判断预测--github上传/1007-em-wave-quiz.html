第一章 矢量分析 预测

梯度 · 散度 · 旋度 · 恒等式 · 散度定理 · 斯托克斯定理

预测 第一章
1. 标量场 u(x,y,z) 的梯度 nabla u 的运算结果是什么类型的量？
  • A. 标量
  • B. 矢量
  • C. 复数
  • D. 张量
答案：B
梯度是标量场对空间坐标的偏导数组成的矢量，输入标量、输出矢量，方向垂直于等值面，沿标量增加最快的方向。
预测 第一章
2. 矢量场 A 的散度 nabla.A 的运算结果是什么类型的量？
  • A. 标量
  • B. 矢量
  • C. 复数
  • D. 矩阵
答案：A
散度 nabla.A = partA_x/partx + partA_y/party + partA_z/partz，结果为标量，表示矢量场在某点发出或汇聚通量的强弱。
预测 第一章
3. 恒等式 nabla x (nabla u) = 0 说明什么？
  • A. 任意标量的散度的旋度恒为 0
  • B. 任意矢量的梯度的旋度恒为 0
  • C. 任意标量的梯度的旋度恒为 0
  • D. 任意矢量的旋度的散度恒为 0
答案：C
nabla x (nabla u) = 0 说明标量场的梯度的旋度恒等于零。重要应用：无旋场可表示为某标量的梯度，从而引入标量位。
预测 第一章
4. 恒等式 nabla.(nabla x A) = 0 说明什么？
  • A. 任意矢量的散度的旋度恒为 0
  • B. 任意矢量的旋度的散度恒为 0
  • C. 任意标量的梯度的散度恒为 0
  • D. 任意矢量的梯度的散度恒为 0
答案：B
nabla.(nabla x A) = 0 说明矢量场的旋度的散度恒等于零。重要应用：无散场可表示为某矢量的旋度，从而引入矢量位。
预测 第一章
5. 散度定理（高斯定理）的数学表达式是？
  • A. oint_C A.dl = int_S (nabla x A).dS
  • B. oint_S A.dS = int_V (nabla.A)dV
  • C. oint_C H.dl = int_S (J + partD/partt).dS
  • D. oint_S D.dS = int_V rho dV
答案：B
散度定理：矢量场散度的体积分 = 该矢量场穿过包围该体积的闭合面的通量。面积分与体积分相互转换。A是Stokes定理。
预测 第一章
6. 斯托克斯定理的数学表达式是？
  • A. oint_C A.dl = int_S (nabla x A).dS
  • B. oint_S A.dS = int_V (nabla.A)dV
  • C. oint_C E.dl = -int_S (partB/partt).dS
  • D. nabla x (nabla x F) = nabla(nabla.F) - nabla^2 F
答案：A
斯托克斯定理：矢量场的旋度在曲面上的面积分 = 该矢量沿曲面边界线的环流。线积分与面积分相互转换。
预测 第一章
7. 连叉公式 nabla x (nabla x F) 等于？
  • A. nabla.(nabla x F) - nabla^2 F
  • B. nabla(nabla.F) - nabla.(nabla F)
  • C. nabla(nabla.F) - nabla^2 F
  • D. nabla(nabla x F) - nabla^2 F
答案：C
连叉公式：nabla x (nabla x F) = nabla(nabla.F) - nabla^2 F。老师特别要求背下来！推导波动方程时极其重要。
预测 第一章
8. 亥姆霍兹定理指出，唯一确定一个矢量场需要知道它的？
  • A. 梯度和旋度
  • B. 散度和梯度
  • C. 散度、旋度和边界条件
  • D. 旋度和边界条件
答案：C
亥姆霍兹定理：任意矢量场由它的散度、旋度和边界条件唯一确定。F(r) = -nabla u(r) + nabla x A(r)。

第二章 电磁场的基本规律 预测

麦克斯韦方程组 · 边界条件 · 本构关系 · 电流连续性方程

预测 第二章
1. 麦克斯韦方程组中反映"变化的磁场产生电场"的方程是？
  • A. nabla x H = J + partD/partt
  • B. nabla x E = -partB/partt
  • C. nabla.B = 0
  • D. nabla.D = rho
答案：B
法拉第电磁感应定律 nabla x E = -partB/partt，物理意义：变化的磁场产生电场（电场的涡旋源）。
预测 第二章
2. 麦克斯韦方程组中反映"传导电流和位移电流都能产生磁场"的方程是？
  • A. nabla x H = J + partD/partt
  • B. nabla x E = -partB/partt
  • C. nabla.B = 0
  • D. nabla.D = rho
答案：A
全电流定律（安培环路定理的推广）：nabla x H = J + partD/partt。位移电流的引入解决了时变情况下安培环路定理的矛盾。
预测 第二章
3. 边界条件中，电场强度 E 的切向分量满足的关系是？
  • A. E1t - E2t = JS
  • B. E1t = E2t
  • C. E1n - E2n = rho_S
  • D. E1t + E2t = 0
答案：B
E的切向分量总是连续：n x (E1-E2)=0, E1t=E2t。记忆技巧：E的切向、B的法向总是连续。
预测 第二章
4. 边界条件中，磁感应强度 B 的法向分量满足的关系是？
  • A. B1n - B2n = JS
  • B. B1n = B2n
  • C. B1n - B2n = rho_S
  • D. B1t = B2t
答案：B
B法向总是连续：n.(B1-B2)=0, B1n=B2n。因为nabla.B=0（磁场无散），磁力线总是闭合曲线。
预测 第二章
5. 电流连续性方程的微分形式是？
  • A. nabla.J = 0
  • B. nabla.J = -partrho/partt
  • C. nabla.J = partrho/partt
  • D. nabla x J = -partrho/partt
答案：B
nabla.J = -partrho/partt，本质反映电荷守恒。恒定电流特例：nabla.J=0（电荷分布不随时间变化）。
预测 第二章
6. 以下哪个是线性各向同性媒质的本构关系？
  • A. D = mu E, B = epsilon H, J = rho E
  • B. D = epsilon H, B = mu E, J = sigma v
  • C. D = epsilon E, B = mu H, J = sigma E
  • D. D = mu H, B = epsilon E, J = sigma E
答案：C
D = epsilon E（电介质）、B = mu H（磁介质）、J = sigma E（欧姆定律微分形式）。D-E同电学量，B-H同磁学量。
预测 第二章
7. 静电场中（part/partt=0），以下哪个结论正确？
  • A. nabla x E != 0，电场有旋
  • B. nabla x E = 0，电场无旋
  • C. nabla.E = 0，电场无散（无源）
  • D. nabla x E = -partB/partt
答案：B
静电场 part/partt=0, nabla x E = 0（无旋），是保守场，可引入标量位 phi 使 E = -nabla phi。

第三章 静态电磁场 预测

静电场标量位 · 恒定电场 · 恒定磁场矢量位

预测 第三章
1. 静电场中引入标量位 phi 的依据是？
  • A. nabla.E = rho/epsilon
  • B. nabla.B = 0
  • C. nabla x E = 0
  • D. nabla x H = J
答案：C
静电场无旋（nabla x E=0），由 nabla x (nabla u)=0 可知存在标量函数 phi 使 E = -nabla phi，电位差与路径无关。
预测 第三章
2. 无源空间（rho=0）中标量电位 phi 满足的方程是？
  • A. nabla^2 phi = -rho/epsilon（泊松方程）
  • B. nabla^2 phi = 0（拉普拉斯方程）
  • C. nabla^2 phi = rho/epsilon
  • D. nabla^2 phi = -mu J
答案：B
有源（rho!=0）时泊松方程 nabla^2 phi = -rho/epsilon；无源（rho=0）时退化为拉普拉斯方程 nabla^2 phi = 0。
预测 第三章
3. 恒定磁场中引入矢量位 A 的依据是？
  • A. nabla x H = J
  • B. nabla.B = 0
  • C. nabla x E = 0
  • D. nabla.D = rho
答案：B
磁场总是无散的（nabla.B=0），由 nabla.(nabla x A)=0 可知存在矢量函数 A 使 B = nabla x A。
预测 第三章
4. 库仑规范的内容是？
  • A. nabla.A = 0
  • B. nabla x A = 0
  • C. nabla.A = -mu epsilon partphi/partt
  • D. nabla^2 A = 0
答案：A
库仑规范 nabla.A=0 用于恒定磁场。C是洛伦兹规范，用于时变电磁场（nabla.A = -mu epsilon partphi/partt）。
预测 第三章
5. 恒定电场出现在什么媒质中？
  • A. 理想介质（sigma=0）
  • B. 导电媒质（sigma != 0）
  • C. 理想导体
  • D. 真空
答案：B
恒定电场出现在导电媒质（sigma!=0）中，维持恒定电流。核心方程：nabla.J=0 和 J=sigma E。
预测 第三章
6. 无电流（J=0）的无源空间中可引入什么来描述恒定磁场？
  • A. 矢量位 A
  • B. 标量磁位 phi_m
  • C. 标量电位 phi
  • D. 库仑规范
答案：B
无传导电流（J=0）时 nabla x H=0（无旋），可引入标量磁位 phi_m, H = -nabla phi_m，类似于静电场标量位。

第四章 时变电磁场 学习通 + 预测

波动方程 · 时谐场复数表示 · 坡印廷矢量 · 能量密度

学习通 第四章
6. 电场能量密度 w_e 的表达式是？
  • A. w_e = J.E
  • B. w_e = H.B/2
  • C. w_e = E.D/2
  • D. w_e = E x H
答案：C
电场能量密度 w_e = 1/2 E.D = 1/2 epsilon E^2。B是磁场能量密度，D是坡印廷矢量。
学习通 第四章
8. 麦克斯韦方程组与波动方程的关系是？
  • A. 二者无关
  • B. 麦克斯韦方程组可推出波动方程
  • C. 波动方程可推出电荷守恒但不能描述波
  • D. 波动方程只用于静电场
答案：B
在理想介质、无源空间中，对麦克斯韦方程组取旋度并利用连叉公式，可推导出波动方程 nabla^2 E - mu epsilon part^2E/partt^2=0。
学习通 第四章
9. 磁场能量密度 w_m 的表达式是？
  • A. w_m = H.B/2
  • B. w_m = rho phi
  • C. w_m = E x H
  • D. w_m = E.D/2
答案：A
磁场能量密度 w_m = 1/2 B.H = 1/2 mu H^2。D是电场能量密度，C是坡印廷矢量。
学习通 第四章
10. 坡印廷矢量 S 的定义是？
  • A. S = E.H
  • B. S = D x B
  • C. S = H x E
  • D. S = E x H
答案：D
S = E x H，方向表示能量传输方向，大小表示单位时间通过单位面积的能量（W/m^2）。
学习通 第四章
12. 当场随时间变化时，空间各点的电磁场能量密度会怎样？
  • A. 只与面积有关
  • B. 随时间改变
  • C. 一定恒定
  • D. 一定为零
答案：B
场随时间变化时，能量密度 w = 1/2 E.D + 1/2 B.H 也随时间改变。
学习通 第四章
13. 对同一电磁场可用不同位函数描述的根本原因是？
  • A. 未规定 A 的散度
  • B. 未规定 E 的旋度
  • C. 未规定媒质的电导率
  • D. 未规定 B 的散度
答案：A
A由B=nabla x A定义，只规定了旋度、未规定散度，因此有无穷多种选择。需要通过选择规范来唯一确定（库仑规范/洛伦兹规范）。
学习通 第四章
14. 坡印廷矢量大小的物理意义是？
  • A. 单位长度相位变化
  • B. 单位体积电荷量
  • C. 单位时间通过垂直于能量传输方向单位面积的电磁能量
  • D. 单位面积电导率
答案：C
坡印廷矢量大小 = 通过垂直于能量传输方向单位面积的功率（W/m^2）。方向为电磁能量的传输方向。
学习通 第四章
15. 时谐电磁场是指场源和场量随时间作什么变化？
  • A. 线性变化
  • B. 指数增长
  • C. 阶跃变化
  • D. 时谐（正弦/余弦）变化
答案：D
时谐电磁场按正弦或余弦规律变化，复数表示中隐含时间因子 e^{j omega t}，part/partt 替换为 j omega。
学习通 第四章
19. 在时谐情况下，可用什么替代时间微分算子 part/partt？
  • A. omega^2
  • B. j omega
  • C. eta
  • D. k
答案：B
时谐电磁场中，时间因子为 e^{j omega t}，时域微分 part/partt 在复数域中替换为 j omega。
学习通 第四章
20. 引入矢量位 A 和标量位 phi 的主要意义是？
  • A. 使场量不再随时间变化
  • B. 消除电荷密度
  • C. 使媒质成为理想导体
  • D. 辅助时变电磁场问题的分析
答案：D
将求解电场磁场的问题转化为求解位函数的问题，再通过位函数导出场量，简化时变电磁场问题的分析过程。
学习通 第四章
24. 时谐场中二次式的瞬时表达式可以直接把复数形式的场量代入相乘。
  • A. 对
  • B. 错
答案：B（错）
复数场量已省略 e^{j omega t} 时间因子，直接相乘会丢失时间信息。需用瞬时表达式计算或用复数量取实部后再运算。
预测 第四章
P1. 推导波动方程时核心使用的数学公式是？
  • A. 散度定理
  • B. 斯托克斯定理
  • C. 连叉公式 nabla x (nabla x F) = nabla(nabla.F) - nabla^2 F
  • D. 亥姆霍兹定理
答案：C
步骤：对nabla x E=-partB/partt取旋度，用连叉公式展开，结合nabla.E=0，代入nabla x H=partD/partt，得到波动方程。
预测 第四章
P2. 平均坡印廷矢量 Sav 的复数表达式是？
  • A. Sav = E x H
  • B. Sav = 1/2 Re(E x H)
  • C. Sav = 1/2 Re(E x H*)
  • D. Sav = 1/4 Re(E x H*)
答案：C
Sav = 1/2 Re(E x H*)，H* 是复共轭。老师说肯定会考坡印廷矢量和平均坡印廷矢量！两种计算方法都要掌握。
预测 第四章
P3. 沿+z传播的均匀平面波，已知 E = ex E0 cos(omega t - kz)，则 H = ？
  • A. H = (1/eta) ex x E
  • B. H = (1/eta) ez x E
  • C. H = eta ez x E
  • D. H = (1/eta) ey x E
答案：B
H = (1/eta) ez x E，大小 |H| = |E|/eta，方向由右手螺旋定则决定。沿-z传播则 H = -(1/eta) ez x E。
预测 第四章
P4. 洛伦兹规范的内容是？
  • A. nabla.A = 0
  • B. nabla.A = -mu epsilon partphi/partt
  • C. nabla.A = mu epsilon partphi/partt
  • D. nabla x A = 0
答案：B
洛伦兹规范 nabla.A = -mu epsilon partphi/partt 用于时变场，使位函数满足达朗贝尔方程（非齐次波动方程）。

第五章 均匀平面波 学习通 + 预测

亥姆霍兹方程 · 传播特性 · 波阻抗 · 波数 · 极化

学习通 第五章
1. 理想介质中的本征阻抗 eta 等于？
  • A. sqrt(epsilon/mu)
  • B. sqrt(mu epsilon)
  • C. sqrt(mu/epsilon)
  • D. 1/sqrt(mu epsilon)
答案：C
eta = sqrt(mu/epsilon)，真空中 eta0 = sqrt(mu0/epsilon0)=120pi=377O。B是波数k，D是相速vp的倒数。
学习通 第五章
3. 两同频率、同方向(+z)的线极化波，极化方向垂直(x与y)，振幅相同、相位差+pi/2时，合成波为？
  • A. 左旋圆极化波
  • B. 右旋圆极化波
  • C. 线极化波
  • D. 椭圆极化波
答案：B
振幅相等+相位差pi/2=圆极化。沿+z传播，+pi/2(y超前x)按右手螺旋=右旋。老师强调圆极化左/右旋判断是高频考点！
学习通 第五章
4. 理想介质中沿+z传播的均匀平面波，E和H与传播方向的关系是？
  • A. 都平行于传播方向
  • B. 都垂直于传播方向（TEM波）
  • C. 磁场平行、电场垂直
  • D. 电场平行、磁场垂直
答案：B
均匀平面波是横电磁波（TEM波），E、H和传播方向ek三者两两垂直，右手螺旋关系 E x H // ek。
学习通 第五章
7. 理想介质中均匀平面波的相速取决于？
  • A. 传播距离
  • B. 媒质参数 mu 和 epsilon
  • C. 电场振幅
  • D. 磁场振幅
答案：B
vp = omega/k = 1/sqrt(mu epsilon)，只取决于媒质的 mu 和 epsilon，与波的振幅和频率（理想介质中）无关。
学习通 第五章
11. 沿+z方向传播的电磁波，其空间相位因子是？
  • A. e^{-omega t}
  • B. e^{-jkz}
  • C. e^{+jkz}
  • D. e^{+alpha z}
答案：B
沿+z传播：E = E0 e^{-jkz}。负号=正方向传播；沿-z传播用 e^{+jkz}（正号=负方向）。不要搞反！
学习通 第五章
16. 平面波是指等相位面为什么形状的电磁波？
  • A. 闭合曲面
  • B. 圆柱面
  • C. 无限大平面
  • D. 任意曲面
答案：C
平面波：等相位面是无限大平面的电磁波。均匀平面波进一步要求等相位面上振幅和方向处处相同。
学习通 第五章
17. "均匀"是指等相位面上电场和磁场的什么保持不变？
  • A. 波长和相速
  • B. 频率和周期
  • C. 方向和振幅
  • D. 电荷和电流
答案：C
"均匀"的含义：等相位面上电场和磁场的振幅相等、方向相同，不随位置变化。

综合典型题

来自总复习资料的典型例题

总复习 综合
已知均匀平面波沿+z方向传播，电场强度 E = ex 100 sin(omega t - beta z) + ey 200 cos(omega t - beta z) V/m，该波的极化状态为？
  • A. 线极化波
  • B. 圆极化波
  • C. 椭圆极化波
  • D. 无极化
答案：C
Ex=100 cos(omega t-beta z-pi/2), Ey=200 cos(omega t-beta z)。振幅Exm=100, Eym=200（不等），相位差Delta phi=0-(-pi/2)=pi/2。振幅不等+pi/2相位差=椭圆极化。
